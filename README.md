# 🐶 Robot Dog 3D

## 📌 Project Overview
This project presents a 3D mechanical design of a robot dog. A web interface was developed to display the model interactively, allowing users to rotate, zoom, and explore the design.

## 🌐 Web Interface
A local web page was created to display the 3D model:

http://localhost:8080/robot-dog-3d.html

The page includes:
- Interactive 3D visualization of the robot dog.
- Rotate and zoom controls.
- Walking animation button.
- Download buttons for OBJ and STL files.
